About BioSearch

Project Overview

BioSearch is a comprehensive search engine designed specifically for biological data. It allows researchers, students, and biologists to search across multiple biological databases simultaneously from a single user interface.

The project aims to simplify the process of finding information about genes, proteins, RNA, and other biological data by providing a unified search experience across various specialized databases.

Key Features

  • Unified Search: Search across multiple biological databases with a single query
  • Real-time Results: Fast asynchronous API requests to multiple databases
  • Organized Results: Tabbed interface for browsing results by database
  • Direct Links: Quick access to original database entries
  • Export Functionality: Save search results as JSON for further analysis
  • Responsive Design: Works on desktop and mobile devices
  • Caching System: Faster repeat searches with intelligent caching

Technical Implementation

BioSearch is built with the following technologies:

  • Backend: Python with Flask web framework
  • API Integration: Aiohttp for asynchronous HTTP requests
  • Frontend: HTML, CSS, JavaScript with Bootstrap
  • Caching: Simple in-memory caching system for improved performance

The application uses asynchronous programming techniques to query multiple databases in parallel, providing faster results to users.

How to Use

  1. Enter a biological search term in the search box on the homepage
  2. Select which databases you want to search
  3. Click the "Search" button to run your query
  4. Browse the results organized by database in the tabbed interface
  5. Click on any result to visit the original database entry
  6. Use the export button to save results for further analysis
